## Дополнительные возможности (опционально)

Можно получить расширенный анализ:

```python
subprocess.run(["python", "RUN_ANALYSIS.py", "outputs/user_123"])
```

Создаст 3 файла:
1. `analysis_result.json` - данные с оценками
2. `REPORT.txt` - текстовый отчёт
3. `analyzed_user_123.mp4` - видео с визуализацией оценок

### analysis_result.json

```json
{
  "figure": "FootChange",
  "confidence": 0.896,
  "overall_score": 85.2,
  "scores": {
    "technique": 76.7,
    "timing": 71.9,
    "balance": 97.8,
    "dynamics": 100.0,
    "posture": 79.7
  },
  "errors_count": 129
}
```

### analyzed_*.mp4

Видео с наложенными:
- Скелетом танцора
- Названием фигуры и уверенностью
- Оценками по 5 метрикам
- Цветными индикаторами
- Подсветкой ошибок

Можно показать пользователю в видеоплеере.
